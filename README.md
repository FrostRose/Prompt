你现在有三个工具。它们解决的是**分析过程中三个不同阶段会出错的地方**。

---

**第一个会出错的地方：你问了一个烂问题。**

比如你问"AI会毁灭人类吗"——这个问题里"AI"是什么、"毁灭"是什么、"人类"是什么，全都是滑动的，没有固定的指向。你拿这个问题去做再严密的分析，得到的也是垃圾，因为垃圾进垃圾出。

**Reasoning Engine v4.2解决这个问题。** 它在你做任何分析之前，先把你的问题解剖一遍：每个关键词的指向是固定的吗？前提和结论之间有真实的因果关系，还是只是看起来有关？它会告诉你这个问题是[INTACT]（可以分析）、[DEGRADED]（需要先收窄）、还是[RECONSTRUCT]（这个问题本身坏了，先重建问题再说）。

**一句话：先用这个确认你在问一个值得回答的问题。**

---

**第二个会出错的地方：你只想到了显见的论点。**

你知道该问什么了，但你的分析会自然地集中在最明显的两个维度——通常是"技术能不能做到"和"钱够不够"。真正把事情搞砸的往往是第三个维度，一个和前两个完全无关的变量，比如保险公司不给承保、监管机构突然介入、关键原材料的供应链根本不存在。

**EIDOLON v7.0解决这个问题。** 它强制生成三个敌对视角：Alpha（乐观派）、Beta（物理/结构现实主义）、Gamma（黑天鹅破坏者）。Gamma的唯一任务是找到一个让Alpha和Beta的整个辩论变得无关紧要的隐藏变量。只跑Alpha和Beta很容易，Gamma是强制你必须去找的东西。

**一句话：用这个确保你想到了那个会让整个分析框架失效的隐藏变量。**

---

**第三个会出错的地方：你知道该问什么，也想到了所有论点，但执行时开始偷懒。**

数字没有来源、偏差没有声明、结论其实是预设的然后往回找理由、叙事停在"他们对外说什么"没有穿透到"他们对自己承认不了什么"。

**Universal Audit v2.1解决这个问题。** 十条铁律，每一条都针对一个在实际测试里观察到的具体逃跑路径。不是让你想得更好，而是让你没法在执行时偷懒。

**一句话：用这个确保你在执行分析时没有耍花招。**

---

**三个工具的使用顺序：**

```
你有一个问题
    ↓
Reasoning Engine v4.2
→ 问题是否值得分析？如果烂，先重建问题。
    ↓
EIDOLON Phase 1（三智能体辩证法）
→ 找到Alpha、Beta的论点，最重要的是找到Gamma变量
  （那个让前两者辩论失效的隐藏变量）
    ↓
Universal Audit v2.1
→ 把Gamma变量带进来作为Escape Vector候选
→ 用十条铁律完成完整的结构性分析
```

---

**什么时候用哪个单独用？**

只需要检查一个命题是否成立、或者别人给你的论点有没有逻辑漏洞 → **只用Reasoning Engine**。它最轻量，专门处理"这个推理本身有没有问题"。

需要快速头脑风暴、找反直觉的破坏性变量，不需要完整分析 → **只用EIDOLON Phase 1**。五分钟出三个敌对视角，Gamma通常是最有价值的。

已经知道该分析什么，需要完整的、执行质量有保证的战略分析 → **只用v2.1**。这是最重的工具，适合真正需要做出决策的场景。

需要最高质量的完整分析，不能有盲点 → **按顺序全用**。

**一个具体的例子：**

问题：**"我们公司应该现在进入AI咨询市场吗？"**

**第一步，Reasoning Engine：** 这个问题里"AI咨询"的范围是什么？是卖战略建议还是卖实施服务？"现在"是指这个季度还是这一年？"应该"的标准是收入、市场份额还是战略定位？→ [DEGRADED]，把问题收窄为："在2026年底之前，进入'帮助企业内部部署开源大模型'这个细分市场，能否在不亏损的前提下建立可持续的收入来源？"

**第二步，EIDOLON：**
- Alpha说：市场需求爆炸式增长，先进去占位，后发优势巨大
- Beta说：MBB和埃森哲已经在压价，利润空间正在快速收窄，你们没有规模优势
- **Gamma说：你们的目标客户的CTO在未来18个月会被要求削减外部咨询支出，同时内部会招AI工程师——外部咨询市场的客户结构正在消失，不是被竞争对手抢走，是这个需求本身在转为内部化**

**第三步，v2.1：** 带着Gamma的发现进来。Escape Vector不是"找到差异化定位"，而是"AI咨询的需求形态在转变，外部咨询窗口有18个月的有效期，之后这个市场的定义本身会变"。Step 4的终态分类是[TOTAL DISSIPATION]（对于传统咨询形态）或[FRAGMENTED EQUILIBRIUM]（对于转型为嵌入式AI团队外包的形态）。结论是：现在进入要么是在18个月窗口里快进快出，要么从第一天就按照"我们是客户内部AI团队的临时替代品"来定位，而不是"我们是AI战略顾问"。

---

这就是三个工具的完整使用逻辑。Reasoning Engine防止你浪费时间在烂问题上，EIDOLON确保你找到了破坏性变量，v2.1确保你的执行没有漏洞。
